# Copyrighted Media Accessibility Remediation Flowchart

An interactive decision-support tool for instructional designers at public universities who need to remediate copyrighted media (videos, podcasts, music) for accessibility in online courses.

## What This Covers

- **Video** captioning and audio description for copyrighted films, documentaries, and recorded lectures
- **Podcast/Audio** transcription for copyrighted audio content
- **Music** remediation including lyrics, descriptive analysis, and visual representations
- **US Copyright Law** compliance: Fair Use (17 U.S.C. Section 107), TEACH Act (17 U.S.C. Section 110(2)), Chafee Amendment (17 U.S.C. Section 121)
- **Accessibility Law** compliance: ADA Title II, Section 504, Section 508, WCAG 2.1 AA
- **Citation procedures** for remediated content (APA 7th Edition)
- **Title II accommodations** including proactive (universal design) and reactive (individual student) pathways

## Hosting on GitHub Pages

This repository is ready for GitHub Pages deployment:

1. Push this repository to GitHub
2. Go to **Settings > Pages**
3. Under **Source**, select **Deploy from a branch**
4. Set the branch to `main` and the folder to `/docs`
5. Click **Save**

Your flowchart will be live at `https://<username>.github.io/media-accessibility-remediation-flowchart/`

The HTML file is fully self-contained (no external dependencies, no build step required). It works as a single file opened locally or served from any static host.

## Legal Framework

This tool addresses the intersection of these legal requirements:

| Law | Relevance |
|-----|-----------|
| [ADA Title II](https://www.ecfr.gov/current/title-28/chapter-I/part-35) | Requires public universities to provide accessible digital content |
| [2024 Title II Final Rule](https://www.ada.gov/law-and-regs/regulations/title-ii-2024/) | Mandates WCAG 2.1 AA compliance by April 2026 |
| [Section 504](https://www.law.cornell.edu/uscode/text/29/794) | Prohibits disability discrimination in federally funded programs |
| [Fair Use (17 U.S.C. Section 107)](https://www.law.cornell.edu/uscode/text/17/107) | Supports captioning/transcription for accessibility |
| [TEACH Act (17 U.S.C. Section 110(2))](https://www.law.cornell.edu/uscode/text/17/110) | Permits limited use of copyrighted works in online education |
| [Chafee Amendment (17 U.S.C. Section 121)](https://www.law.cornell.edu/uscode/text/17/121) | Allows reproduction in accessible formats for print disabilities |
| [WCAG 2.1 AA](https://www.w3.org/TR/WCAG21/) | Technical standard for web accessibility |
